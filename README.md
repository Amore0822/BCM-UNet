# BCM-UNet

Official implementation of **BCM-UNet: Boundary-Guided Feature Refinement for Remote Sensing Semantic Segmentation**.

## Overview

BCM-UNet is a boundary-guided feature refinement framework for high-resolution remote sensing semantic segmentation. It extends CM-UNet with a Boundary Prediction Module (BPM) and a Boundary-Guided Feature Refinement Module (BGFRM). Boundary labels are automatically generated from semantic segmentation masks without requiring additional manual boundary annotations.
